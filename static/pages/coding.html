CSS3

In recent years CSS has become a very powerful language. Transitions and animations applied to elements, gradually changing from one style to another creates a smooth user experience. CSS could be used to slightly change the size and color of a button on hover, or to fade in a page on load, or even to rotate the loading spinner while the page is loading. HTML

HTML makes up the content of webpages and the latest standard HTML5 contains tons of elements that you would need inside a webpage, ranging from inputs, buttons, svg, images, text, etc...

Angular JS is a platform loved by millions that extends the functionality of HTMl to include conditional statements, templates, data binding, angular directives and much more. It is a platform I often use when developing web apps as it is very scalable and can handle huge data requirements and because it can significantly increase the speed of development.
