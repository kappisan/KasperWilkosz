    In recent years CSS has become a very powerful language. Transitions and animations applied to elements, gradually changing from one style to another creates a smooth user experience.     HTML makes up the content of webpages and the latest standard HTML5 contains tons of elements that you would need inside a webpage, ranging from inputs, buttons, svg, images, text, etc...

    Angular JS is a platform loved by millions that extends the functionality of HTMl to include conditional statements, templates, data binding, angular directives and much more.
